# 부여 오디오 가이드 음원 (장소별)

`/buyeo/:stop` 페이지에서 재생할 **장소별** 오디오 파일을 둡니다.
컨셉: QR 1개 = 사운드 1개. 각 장소(동선상 도착지)마다 사운드 하나.

현재는 **임시 placeholder** 상태입니다. 라이선스 정리 후
퍼블릭도메인 또는 직접 제작한 음원으로 교체하세요.

## 파일명 (장소별)

| 장소 | 라우트 | 파일명 |
|------|--------|--------|
| 장소 1 | `/buyeo/1` | `buyeo-stop1.mp3` |
| 장소 2 | `/buyeo/2` | `buyeo-stop2.mp3` |

## 교체 방법

1. 위 파일명으로 음원을 이 폴더에 추가합니다(파일명을 유지하면 코드 수정 불필요).
2. 또는 새 파일명을 쓰려면 `src/components/Buyeo.tsx` 상단의
   `BUYEO_STOPS[].src` 값만 수정하면 됩니다.

장소 추가도 `BUYEO_STOPS` 배열에 항목 하나만 더하면 됩니다
(예: `stop: 3`, `src: '/audio/buyeo-stop3.mp3'`). 그러면 `/buyeo/3`이 자동 동작합니다.
